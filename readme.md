# TeamFlow SaaS

TeamFlow is a multi-tenant project and task management SaaS built using Django REST Framework.

## Features

- JWT Authentication
- Custom User Model
- Multi-tenant architecture
- Company workspace isolation
- Project and Task management
- Role-Based Access Control (RBAC)

## Tech Stack

Backend:
- Python
- Django
- Django REST Framework

Authentication:
- JWT (SimpleJWT)

Database:
- SQLite (development)

Tools:
- Git
- Postman
